About

PeerScript labs is Peercoin's Area 51 for smart contracts. It is a test ground for the development of specialized Peercoin transaction scripts. Other projects benefit from PeerScript Labs integrating scripts developed and proven on this test ground.

Puzzles

Puzzles are funds locked by a script puzzle. One can lock a certain amount of Peercoins that can be spent by anyone knowing the solution to: x + 5 = 6.
This section contains a set of puzzles that you can solve to earn some Peercoins.
After solving a puzzle, we kindly ask you to create a new puzzle for other people to solve and add it to this project by creating a pull request on github.

Smart Contracts

Smart Contracts are used to lock funds under certain conditions. Choosing these conditions wisely allows for the creation of specialized trustless contracts. The complexity of a contract can vary from a simple escrow transaction to trustless cross chain trading. This section demonstrates how those contracts are built on the Peercoin blockchain.
